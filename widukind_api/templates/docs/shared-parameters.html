Parameters:

limit

The maximum number of results to return

  • Required: false
  • Type: int
  • Default: 1000
  • Example: ?limit=1000

frequency

Frequency filter

  • Required: false
  • Type: str
  • Default: None
  • Example: ?frequency=Q
  • Available values (depend dataset): A, Q, M, W

<dimension_or_attribute>

Dimension or attribute filter

  • Required: false
  • Type: string
  • Default: None
  • Example: ?reference-area=FR (only series with dimension reference-area=FR)
  • Example: ?reference-area=!FR (exclude series with dimension reference-area=FR)
  • Example: ?reference-area=de+au (select DE+AU)
  • Comment: No case

tags

Tags filter

  • Required: false
  • Type: str
  • Default: None
  • Example: ?tags=france
  • Example: ?tags=france+australia
  • Example: ?tags=fr+au
  • Comment: No case
